Glitch Art

In this project, we did a project called "Glitch Art" where we used different programs to create glitch rather than using photoshop.

Our prompt for this project was to create a series of three glitch images using different methods: text edit and audacity.

For my project, I used My Melody and Kuromi pictures because I love their character design and I have an obsession of sanrio.

First picture was created by text edit. Second picture was created by audacity and last was a combination of text edit and audacity.
