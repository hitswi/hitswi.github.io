Net Art

Each students must own a minecraft (JavaScript) in order to do this project.

The prompt for this project was to create an object of cultural significance using class server that professor provided us.

I chose hello kitty for my cultural significance object since she became a phenomenon character in the whole nation and that she is symbolize as empowerment and womanhood in Asian culture.

Not only that, Hello Kitty was the one that shaped me into who I am today.
